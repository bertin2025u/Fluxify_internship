# Week 2 – Fluxify Internship Project

## Project Overview
This project includes:
- Express.js server
- MySQL integration
- REST API with CRUD
- Validation and error handling
- Postman testing

## Repository Structure
fluxify-internship/
└── week2/
    ├── .gitignore
    ├── README.md
    ├── express-server/
    ├── rest-api/
    └── postman/

## Installation
Clone the repository:
git clone <your-repo-link>

Install dependencies:
cd express-server
npm install

cd ../rest-api
npm install

## Required Packages
npm install express morgan mysql2 dotenv
npm install -g nodemon

## Database
Start XAMPP MySQL and create:
CREATE DATABASE online_shop;

## Running
Express Server:
nodemon app.js

REST API:
nodemon app.js

## Features
- Full CRUD
- Validation
- Global error handling
- Postman tested
