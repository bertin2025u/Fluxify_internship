# Postman Collection Guide

## Import Collection
1. Open Postman
2. Click Import
3. Select Shop_API.postman_collection.json

## Tested Endpoints
- GET /products
- GET /products/:id
- POST /products
- PUT /products/:id
- DELETE /products/:id

## Assertions
Each request verifies:
- Correct HTTP status code
- JSON response structure
- Database persistence
